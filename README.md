# 解一元N次方程
n> 此项目已迁移至 [CppWorkspace](https://github.com/BlazeSnow/CppWorkspace/tree/main/SolvingEquations)

1. 此程序用**二分法**、**牛顿迭代法**和**弦截法**解一元N次方程；
2. 第一次使用将自动创建**solving-equations.txt**；
3. 计算内容写在**solving-equations.txt**中；
4. 各数字之间加上空格，输入到文件时按以下格式：

```ansi
f(x) = -x² -2.2x +3

-1 2 -2.2 1 3 0
```
